### Prerequisites

- [STEAM API KEY](https://steamcommunity.com/dev)
- **NodeJS** v16+
- **npm** 8.1.0+

### Installation

```sh
# Open terminal
git clone https://github.com/freakGE/Dota2-HeroCounters-CLI.git
cd Dota2-HeroCounters-CLI
echo "STEAM_API_KEY=${KEY}" >> .env
npm install
```

### Usage

```sh
 node heroCounter.js
```
